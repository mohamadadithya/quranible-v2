# Quranible V2

Quranible is a complete Al-Quran website with transliteration and Indonesian translation.

## Getting Started

### Prequisites

Node JS v16.14

### Installing

```
npm install
```

### Run program locally

* How to run the program
```
npm run dev
```

* How to build
```
npm run build
```

## Features
- [x] Bookmark ayah
- [x] Play ayah audio
- [x] App settings
  * Theme
  * Show/Hide transliteration
  * Ayah audio auto scroll
- [x] PWA
